# SafeNet – Human Trafficking Reporting System

SafeNet is a full-stack application designed to report and track human trafficking cases.

## Tech Stack
- .NET Core Web API
- React
- SQL Server (Stored Procedures)
- Dapper
- GitHub Actions (CI/CD)

## Project Structure

backend/  → .NET API  
frontend/ → React UI  

## Features
- Report human trafficking incidents
- Anonymous reporting
- Dashboard with reports list
- CI pipeline using GitHub Actions
